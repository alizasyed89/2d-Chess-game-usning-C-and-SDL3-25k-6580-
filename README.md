# Overview: 
This project presents the development of a fully functional 2D Chess Game using the C 
programming language and the SDL3 graphics library. The system displays an 8×8 chessboard, 
loads graphical textures for chess pieces, and allows two players to play by following standard 
movement rules. The project demonstrates key programming fundamentals such as arrays, 
enums, loops, conditionals, and event-driven programming.
